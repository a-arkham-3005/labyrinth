Лабиринт
--------

Описание
========

Это изначальная версия игры, разработанная в рамках летней практики 2024 года, цель которой - поиск информации за ограниченное число ходов. Это моя первая разработка в среде Unity. К сожалению, в связи с сильно ограниченным временем на практику (2 недели) и последующей нехваткой желания и времени (из-за учёбы, личных дел и других разработок) её дорабатывать, данный выпуск является предварительным, в котором реализован лишь базовый функционал из того, что описано в документации.

Основная документация по концепции игры, разработанная командой из практикантов при моём участии, лежит в папке docs. Данная игра представляет собой один из модулей игры-обучалки по проектному менеджменту.
